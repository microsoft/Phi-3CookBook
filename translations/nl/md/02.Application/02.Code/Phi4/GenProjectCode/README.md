## **Phi-4-mini-mm gebruiken om code te genereren**

Phi-4-mini zet de sterke codeermogelijkheden van de Phi Family voort. Je kunt Prompt gebruiken om vragen te stellen die gerelateerd zijn aan coderen. Natuurlijk, met de toevoeging van sterke redeneervaardigheden, heeft het nog krachtigere codeermogelijkheden, zoals het genereren van projecten op basis van eisen. Bijvoorbeeld, projecten genereren op basis van eisen, zoals:

### **Eis**

Maak een Shopping Cart App

- Maak een API Rest met de volgende methoden:
    - Haal een lijst met bieren op met behulp van paginering (offset en limiet).
    - Haal bierdetails op via id.
    - Zoek naar bier op naam, beschrijving, slogan, voedselcombinaties en prijs.
- Maak een lijst van producten op de hoofdpagina.
    - Maak een zoekbalk om producten te filteren.
    - Navigeer naar de beschrijvingspagina wanneer de gebruiker op een product klikt.
- (Optioneel) Een slider om producten te filteren op prijs.
- Maak een winkelwagentje.
    - Voeg producten toe aan het winkelwagentje.
    - Verwijder producten uit het winkelwagentje.
    - Bereken de totale prijs van de producten in het winkelwagentje.
